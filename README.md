## 📦 安装运行

### 环境要求
- 现代浏览器（Chrome 90+、Firefox 88+、Safari 14+）
- 本地HTTP服务器（可选）

### 快速开始
```bash
# 克隆项目
git clone [repository-url]

# 进入项目目录
cd 官网

# 启动本地服务器
python -m http.server 8000
# 或使用其他HTTP服务器

# 访问页面
http://localhost:8000
```

### CSS构建（如需修改样式）
```bash
# 安装TailwindCSS（如需自定义）
npm install -D tailwindcss
npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```
